课程说明

这门课程的内容:　Terraform 在 DevOps 工程师中越来越受欢迎是基于基础设施即为代码的原则。 Terraform 允许任何人使用 AWS、 Google Cloud 和 Kubernetes 从头开始自由地建构整个云端基础设施。

这门课程教授那些希望扩展他们的开发和营运角色的人，包括掌握 Terraform 作为他们技能的一部分。 课程从 Terraform 基础到 AWS Instance 的高端用法示范，Kubernetes，Helm，以及 Ansible 和 Google Cloud。 每个部分都是实际操作的，包括视频、示范和练习，你可以尽快立即开始自动化你自己的基础设施。

你将从这门课程中学到什么:

  • 很好地理解了 Terraform 和 Terraform 代码
  • 能够在 AWS 中部署可扩展的基础设施
  • 能够创建复杂的 Terraform 样板
  • 在云端中部署高可用性的应用程序
  • 更好地了解 DevOps 和基础设施服务。
  • 了解使用 Terraform 做基本 kubernetes 部署
  • 能够管理和部署  kubernetes

在课程快结束时，还有两个额外的部分:

使用 AWS 与 Terraform: 这一部分真正深入地使用 Terraform 与 AWS 提供的许多受欢迎的服务。 在这个过程中，我们探索从 VPCS 到 Lambda serverless 的所有东西，解释它们是如何运作和如何集成的。

Terraform 与 Kubernetes & Helm : 这个部分转向 Google Cloud，专注于在 Kubernetes 创建一个高度可扩展的 MongoDB 集群。 它教授 Kubernetes 和 Helm 的基本知识，以及将这两者与 Terraform 连接起来。
