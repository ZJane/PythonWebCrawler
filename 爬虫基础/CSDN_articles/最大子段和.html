N个整数组成的序列a11,a22,a33,…,ann， 求该序列如aii+ai+1i+1+…+ajj的连续子段和的最大值。当所给的整数均为负数时和为0。

例如：-2,11,-4,13,-5,-2，和最大的子段为：11,-4,13。和为20。

Input

第1行：整数序列的长度N（2 <= N <= 50000) 
第2 - N + 1行：N个整数（-10^9 <= Aii <= 10^9）

Output

输出最大子段和。

Sample Input

6
-2
11
-4
13
-5
-2

Sample Output

20
