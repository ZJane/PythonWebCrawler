Httpd和smb的配置

2018年08月07日 20:40:47
阅读数：1

Httpd 配置目录映射

Alias /client "/home/data/client/"
<Directory "/home/data/client/">
    Options Indexes MultiViews FollowSymLinks
    AllowOverride None
    Order allow,deny
    Allow from all
</Directory>

 

Smb配置目录访问

[client]
        comment = Client Dir
        path = /home/data/client/
        writeable = yes
        public = yes
        browseable = yes

无需用户密码访问，记得开启

security = share

如果不能往共享目录写文件，记得设置目录权限

chmod -R 777 client
